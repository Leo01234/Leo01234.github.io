    RTL9210 主控固态硬盘盒刷不休眠固件

    Leo012342024年10月5日大约 4 分钟学习RTL9210固态硬盘盒不休眠固件
    此页内容
    • 起因
    • 解决方法
    • 总结
    • 参考资料

    起因

    Dell G15 5520 游戏本没有第二个硬盘位，本身自带的硬盘只有 500G。随便装点东西就不够用了，此时外接一个固态硬盘就是很好的选择。使用雷电接口接固态硬盘盒，读写速度不是问题，可以直接将软件/游戏装在外接的固态硬盘里使用。

    然而使用一段时间（不到一周），发现硬盘通电次数涨到了 500 多。

    查看方法

    一个软件 CrystalDiskInfo，在图吧工具箱里也有

    这属于不正常现象，很有可能会造成硬盘损坏。

    解决方法

    刷不休眠固件。实际上不用动原本的固件，只需要修改配置文件，然后只刷配置文件就可以了。

    需要用到软件 UTHSB_MPtool_Lite.exe。在这里可以下载固件，自带了此软件。实际上不需要用到里面的固件，只需要里面的刷固件的软件。

    1. 连接固态硬盘盒。我是使用的雷电接口连接，据说刷固件时使用普通 USB 接口（Type-A）更好。我硬盘盒中是放着硬盘的，据说放不放都可以。
    2. 打开 UTHSB_MPtool_Lite.exe 软件，首先 Dump -> Show Device Info，保存 Dump 的信息。这就是目前固态硬盘盒各项配置的信息。如下所示：

    1. 修改软件自带的配置文件 RTL9210.cfg，使其与 Dump 出的配置一致。值为 n/a 的忽略。

    关闭休眠功能

    DISK_IPS_THRES 改为 0

    VID = 0xbda
PID = 0x9210

UART_DBG_PIN = 5
SUSPEND_DISK_OFF = 0x7
    1. Settings 菜单，取消 Firmware Update 与 Disk Format 前面的 √，只选择 PG。
    2. 确保 Device Select 是想要修改的设备，Configure 是前面编写的配置文件，然后点 Update Device 等执行结束就可以了。
    3. 最后可以再次 Dump 一下，看是否修改为了需要的值。

    总结

    修改完成后，使用几个月后通电次数只增加了一点（一百多）。确实有效。
